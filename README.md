# Random Word Generator

A website and Chrome extension for generating random words and passphrases.

Word count, word complexity, capitalization, separator, and length are configurable. Processed locally in the browser, nothing is saved.

View here: [https://unjaked.github.io/random-word-generator/](https://unjaked.github.io/random-word-generator/)

## Install as Chrome Extension

1. Clone/download this repo locally.
2. Open `chrome://extensions`.
3. Enable **Developer mode** toggle in the top right.
4. Select **Load unpacked**.
5. Choose this project folder.

## Word lists

- [EFF Short (Simple) Wordlist #1](https://www.eff.org/document/eff-short-wordlist-passphrases-1): 1,296 simple words
- [EFF Long (Complex) Wordlist](https://www.eff.org/dice): 7,776 words
